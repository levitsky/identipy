Pyteomics-based search engine.

# How to install #

```
#!shell

$ hg clone https://bitbucket.org/levitsky/identipy
$ cd identipy
$ pip install .

```

# How to use #

For help on command-line usage, run:


```
#!shell

$ identipy --help
```
